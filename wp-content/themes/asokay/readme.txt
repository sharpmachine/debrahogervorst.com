Author: Andreas Viklund

Requires at least: WP 3.0
Tested up to: WP 3.2.1

A simple and visually clean theme by Andreas Viklund.

== Description ==

Based on a classic two-column layout, Asokay provides a fast-loading and visually clean starting point for both blogs and simple websites. It is made to be easy to work with, while not utilizing any of of the more advanced features that are common in magazine- or portfolio-styled themes.

== Installation ==

You can install the theme through the WordPress installer under <strong>Themes &rarr; Install themes</strong> by searching for it.
Alternatively you can download the file from here, unzip it and move the unzipped contents to the <code>wp-content/themes</code> folder
of your WordPress installation. You will then be able to activate the theme.
