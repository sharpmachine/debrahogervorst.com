== Reference Theme Read Me ==

The theme features the following additional widget areas:
1) Top Banner - use a text widget to display a banner
2) Menu Widget Area - use a text widget to display text links under the navigation menu
3) First Header Widget Area - use a text widget to display conntet in the top left content box
4) Second Header Widget Area - use a text widget to display content in the middle top content box
5) Third Header Widget Area - use a text widget to display content in top right content box
6) Content Bottom Widget Area - use a text widget to diaplay an additional banner below page content

Theme homepage: http://supersitewebhosting.com/free-wordpress-theme-reference
